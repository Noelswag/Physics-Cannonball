# Controls

A: Move Left
D: Move Right
W: Jump
SPACE: Fire cannon ball

# Debug options

F2: Change movement option (MOMENTUM, IMPULSE AND ACCELERATION NOT IMPLEMENTED YET)
F3: Change jump option (MOMENTUM, IMPULSE AND ACCELERATION NOT IMPLEMENTED YET)

1: Euler
2: Euler Sympletic
3: Velocity Verlet
